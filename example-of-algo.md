**Задача** 

Написать скрипт, который будет перезапускать сетевой интрефейс (down - up; команды есть в уроке от 29.10)
название сетевого интерфейса должен ввести пользователь как позиционный аргумент

---

1) На вход через позиционный аргумент подаем скрипту имя интерфейса и через второй позиционный аргумент подаем действие (up/down);
в результате - интерфейс перешел в статус UP/DOWN; (? надо ли делать вывод об успехе/провале)
2) Что нужно для решения задачи
- проверить, что было передано 2 аргумента и второй это up/down
    - если не два аргумента, то вывести подсказку как запустить и завершить скрипт
    - если не up/down, то вывести подсказку о некорректном действии и завершить скрипт
- проверить есть ли такой интерфейс на ВМ
    - требуются права root -> перед проверкой наличия интерфейса - проверить что скрипт запущен с `sudo`
    - получить список всех интерфейсов
    - проверить, есть ли среди них введенный пользователем
- если есть, то в зависимости от action выполнить команду по переводу в соответствующий статус (up/down)
- если нет, то вывести сообщение о некорректном имени интерфейса
3) Команды
- **проверить, что было передано 2 аргумента**
```bash
if [ "$#" -ne 1 ]; then
    echo "Illegal number of parameters"
fi
```
где `$#` - спец переменная, хранящая _количество_ переданных скрипту аргументов
- проверить что второй аргумент up/down
```bash
if [ "$2" == "up" ] || [ "$2" == "down" ]; then
  echo "The strings are equal."
else
  echo "The strings are not equal."
fi
```
- завершить скрипт `exit 1`
- проверить есть ли такой интерфейс на ВМ
    - требуются права root -> перед проверкой наличия интерфейса - проверить что скрипт запущен с `sudo`
```bash
if [ $(id -u) -ne 0 ]; then
  echo "Need root privileges"
  exit 1
fi
```
  - получить список всех интерфейсов `ip -o link show | awk -F: '{print $2}' | tr -d ' '`
  - проверить, есть ли среди них введенный пользователем
```bash
if_exists=false
for i in "${arr[@]}"
do
   echo "$i"
   if [ "$i" == "$1" ]; then
     if_exists=true
     breack
   fi
done
```
  - если есть, то в зависимости от action выполнить команду по переводу в соответствующий статус (up/down)
```bash
sudo ip link set $i $2
```
  - если нет, то вывести сообщение о некорректном имени интерфейса
```bash
echo "Incorret input"
exit 1
```
